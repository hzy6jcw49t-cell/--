## 主な実装

- **レスポンシブ対応** — 768px以下でレイアウト・ボタン位置を変更
- **MENUドロワー** — 開閉・オーバーレイ・Escキー対応
- **スクロール演出** — フェードイン・スムーススクロール
- **実用リンク** — `tel:` / `mailto:` / Instagram
- **Figma準拠** — PC・スマホの文言・余白・構成を統一

## こだわった点

**デザイン再現**  
お部屋セクションでは、スマホ表示時にボタンをギャラリー途中に配置するなど、デザインカンプに合わせた分岐を実装。

**和の世界観**  
和紙背景・筆文字タイトル（Yuji Boku）・ゴールドボタンで統一感を演出。

**コードの整理**  
メディアクエリの統合、未使用スタイルの削除など、保守しやすい構成を意識。

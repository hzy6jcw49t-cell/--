## プロジェクト概要

京都の山あいに佇む温泉旅館「松風」の世界観を、和紙テクスチャ・明朝体・余白の取り方で表現した静的サイトです。PC・スマホ両方のデザインカンプをもとに実装しています。

## 主な機能・実装

- **レスポンシブ対応**（768px 以下でレイアウト・ボタン位置を変更）
- **MENUドロワー**（開閉・オーバーレイ・Escキーで閉じる）
- **スクロール演出**（セクションのフェードイン、スムーススクロール）
- **実用的なリンク**（`tel:` / `mailto:` / Instagram）
- **Figmaデザイン準拠**（PC・スマホの文言・余白・構成を統一）

## こだわった点

### デザイン再現
FigmaのPC版・スマホ版を照合しながら、セクションごとにレイアウトを調整しました。例えばお部屋セクションでは、スマホ表示時にボタンをギャラリー途中に配置するなど、デザインカンプに合わせた分岐を実装しています。

### 和の世界観
和紙背景（`paper.jpg`）、筆文字タイトル（Yuji Boku）、ゴールドボタン（`#e6dccb`）で、旅館らしい静けさと統一感を意識しました。

### コードの整理
メディアクエリの統合、未使用スタイルの削除など、保守しやすいCSS構成を心がけました。
